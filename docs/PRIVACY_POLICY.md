# Privacy Policy

OwlFly currently stores:

- High score
- Basic settings (such as mute and reduced motion)
- Other local gameplay preferences if added in the same local-only model

All storage is currently local to your device through browser or app storage mechanisms such as `localStorage`.

## Data collection

OwlFly does not currently:

- collect personal data
- transmit gameplay data to external servers
- require account creation
- share personal information with third parties

## Ads and analytics

The current internal-testing direction assumes no live personal-data collection.

If future versions add:
- ads
- analytics
- leaderboards
- cloud saves
- remote feature flags

then this policy must be updated before public release to reflect:
- what data is collected
- why it is collected
- where it is stored
- which third parties, if any, receive it

## Contact / update note

This policy should be reviewed again before:
- open testing
- production release
- any backend or SDK integration
