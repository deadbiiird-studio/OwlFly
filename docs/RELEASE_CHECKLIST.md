# Release Checklist (OwlFly)

## Core Gameplay

- [ ] No crashes during full run cycles
- [ ] Collision feels fair and consistent
- [ ] Gap sizes are readable with new sprites
- [ ] Tornado bottom hitboxes feel accurate
- [ ] Cloud top visibility is clear

---

## Visual Identity

- [ ] Clouds render correctly (no stretch artifacts)
- [ ] Tornado animation cycles smoothly
- [ ] No missing textures / broken paths
- [ ] Owl animation synced with gameplay

---

## UI / UX

- [ ] Game fills full screen on device
- [ ] HUD properly anchored (not following owl incorrectly)
- [ ] Menu → Game → Game Over flow works cleanly
- [ ] No duplicate UI elements

---

## Audio

- [ ] No overlapping sounds (especially game over)
- [ ] Audio triggered once per event
- [ ] Audio loads correctly on mobile

---

## Build / Platform

- [ ] `web/game.js` builds successfully
- [ ] Assets copied to Capacitor correctly
- [ ] App icon appears correctly on device
- [ ] App runs fullscreen on Android

---

## Testing

- [ ] Internal testers can install and run
- [ ] Feedback channel exists (form/email)
- [ ] Major bugs logged and triaged

---

## Store Prep

- [ ] Icon (512x512)
- [ ] Feature graphic
- [ ] Screenshots
- [ ] Description

---

## Final Gate

- [ ] Game feels intentional (not clone-like)
- [ ] At least one standout visual/mechanic present
