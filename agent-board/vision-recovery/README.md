# OwlFly Vision Recovery Agent Pack

## Purpose

This folder contains the prompt rooms used to carry OwlFly to completion without losing the original vision.

OwlFly is not being completed as a generic Flappy-style clone. It is being completed as a stylized skyline flyer with:

- San Antonio-inspired bottom buildings
- upper cloud hazards
- readable building/cloud gap logic
- a right-facing owl with clear motion identity
- a fracture transition
- a true glide mode
- larger-than-owl rings
- Android proof
- build-in-public documentation

## Core Rule

Every change must answer:

> Does this make OwlFly more unmistakably OwlFly?

If yes, continue.

If no, backlog it.

## Completion Standard

OwlFly is ready only when it is:

- playable
- readable
- visually cohesive
